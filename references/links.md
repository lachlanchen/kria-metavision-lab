## What this folder contains

- `~/Projects/kria-kv260-starter/fpga-projects`: Prophesee FPGA build tree
- `~/Projects/kria-kv260-starter/petalinux-projects`: KV260 PetaLinux stack (branch `kv260-2022.2`)
- `~/Projects/kria-kv260-starter/linux-sensor-drivers`: Linux drivers
- `~/Projects/kria-kv260-starter/zynq-video-drivers`: Zynq video stack
- `~/Projects/kria-kv260-starter/event-vitisai-app`: optional ML application repo (README/sparse checkout)
- `~/Projects/kria-kv260-starter/references/kv260-prophesee-resources.md`: local captured notes from provided links
- `~/Projects/kria-kv260-starter/references/links.md`: link list
- `~/Projects/kria-kv260-starter/references/kv260-manual-downloads.md`: manual download checklist
